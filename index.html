Shadow Hunter

an OnChain PvP Game

How to Play

  1. Players mint "Assassin" NFTs which can then be used to "assassinate" other player's NFTs
  2. Players select their target NFT by inputting the target's ETH address a to the game's smart contract in order to initiate the assassination.
  3. Once the smart contract confirms that the assassination is valid, the target NFT is destroyed during the next block to prevent any double attacks.
  4. The game ends when there is one Assassin left and the owner of that assassin gets the allocated prize.
  5. There are multiple mechanism to make this game more strategic, challenging and fun. You can read about them here.
